# 05. Encounters and Counterplay

## 1. 설계 목적

원정마다 같은 빌드와 같은 준비가 정답이 되지 않도록 한다. 플레이어는 지역 정보와 예상 위협을 바탕으로 준비하고, 현장에서 드러난 변형에 맞춰 대응한다.

## 2. 상성 철학 — LOCKED

하드 카운터보다 소프트 카운터를 사용한다.

권장 체감 난도:

- 준비가 매우 잘 맞음: 기준 난도의 60~70%
- 보통 준비: 100%
- 불리한 준비: 140~170%

특정 장비가 없으면 사실상 처치 불가능한 구조는 지양한다.

## 3. 대응 축

### 공격 방식

- 단일 대상
- 광역
- 관통
- 지속 피해
- 고빈도 저피해
- 저빈도 강타
- 설치형
- 소환형

### 이동과 공간

- 좁은 통로에 강한 적
- 넓은 공간에 강한 적
- 돌진형
- 순간이동형
- 벽 관통형
- 벽 생성형
- 굴착형

### 지속 시간

- 시간이 지나면 광폭화
- 장기전에서 약화
- 잡몹을 흡수해 회복
- 각성도에 따라 강화

### 자원과 위험

- 광맥을 먹고 강화
- 보물을 훔침
- 채굴 소음에 반응
- 특정 지형에서 재생

## 4. 네임드 생성 방식 — LOCKED

완전 자유 조합이 아니라 수제 조합 풀을 사용한다.

```text
기본 개체
+ 해당 개체에 허용된 변형 1개
+ 검증된 강화 특성 0~1개
+ 고유 이름과 보상
```

### 조합 금지 예시

- 자동 공격을 강제로 자해시키는 강한 피해 반사
- 순간이동 + 무적 + 고속 재생
- 화면 밖 장거리 공격 + 지속 은신
- 좁은 공간 강제 + 광역 즉사

## 6. 현장 대응 수단 — LOCKED

준비가 어긋나도 비용을 지불해 수정할 수 있어야 한다.

- 임시 탄약 또는 코팅 제작
- 무기 하나 교체
- 업그레이드 재선택
- 유물 즉시 장착
- 보스 흔적 조사로 약점 공개
- 서브 던전 공략으로 대응 아이템 획득
- 전투 공간 사전 채굴

대가는 다음 중 하나 이상이다.

- 시간
- 각성도
- 보물 또는 자원
- 장비 슬롯
- 추가 위험

## 7. 후퇴와 잔존 위협

네임드를 처치하지 않고 떠나면 다음 캠페인 요소로 남을 수 있다.

- 해당 지역 희귀 자원 감소
- 일반 적 강화
- 네임드 토벌 임무 생성
- 다음 원정에서 재등장
- 네임드가 새로운 특성을 획득 **DEFERRED**
