# 08. World, Art, and Audio

## 1. 세계관 방향 — DRAFT

지하세계는 오래전에 추락하거나 죽은 거대한 존재의 신체와 고대 문명의 광산 시설이 뒤섞인 공간이다.

- 광석은 결정화된 피, 뼈, 신경, 장기와 유사한 물질
- 봉인석은 심층부의 핵심 존재를 억누르는 장치
- 플레이어는 자원 채굴과 보물 회수를 명목으로 심층부를 탐사
- 최종 목표는 핵심 유물을 훔쳐 지상으로 탈출하는 것

세계관은 게임 시스템을 설명하는 역할을 우선하며, 세부 설정은 프로토타입 이후 확장한다.

## 2. 추천 아트 방향 — LOCKED / DRAFT 혼합

### 기술 구조 — LOCKED

- 2D 타일맵 기반
- 셀 단위 파괴형 지형
- 2D 충돌과 이동
- 스프라이트 캐릭터와 몬스터
- 직교 투영 카메라

### 시각 표현 — DRAFT

- 2.5D 인상을 주는 쿼터뷰 또는 가짜 높이
- 벽 상단·정면 분리 표현
- 2D 광원, 그림자, 발광 광맥
- 어두운 광산 디오라마 분위기
- 캐릭터보다 환경과 VFX에 분위기 집중

## 3. 아트 제작 원칙

- 핵심 스타일 기준 에셋 하나를 먼저 선정
- 프로젝트 전체 팔레트 통일
- 픽셀 밀도와 외곽선 규칙 통일
- 무기와 VFX를 캐릭터 애니메이션에서 분리
- 4방향 이하 애니메이션으로 시작
- AI 생성물은 초상화·삽화·보물 카드·컨셉 시안에 우선 사용
- 반복 사용되는 플레이 스프라이트는 에셋과 수작업 보정 중심

## 4. AI 사용 권장 영역

- 광부와 NPC 초상화
- 보물 일러스트
- 캠페인 사건 삽화
- 로딩 화면
- 컨셉 아트
- 배경 텍스처 초안

## 5. AI 사용 주의 영역

- 동일 캐릭터의 방향별 애니메이션
- 타일 연결 규칙이 필요한 완성 타일셋
- 일관된 UI 아이콘 전체 세트
- 장비별 캐릭터 변형
- 핵심 보스의 반복 애니메이션

## 6. 오디오 방향 — DRAFT

오디오는 시야 밖 위험과 지형 정보를 전달하는 기능을 가져야 한다.

- 채굴 재질별 타격음
- 네임드 접근 징후
- 공명도 상승 음향
- 비밀방 근처의 미세한 음향
- 탈출 단계의 음악 변화
- 광부 부상과 장비 고장의 명확한 피드백

## 7. 접근성과 가독성

- 적 역할은 색상뿐 아니라 실루엣과 이동 방식으로 구분
- 광원 제한 상황에서도 핵심 위험 표시
- 자동 공격 타깃 규칙을 UI로 설명
- 피격·방어·무효·장갑 반응을 다른 VFX와 사운드로 구분
