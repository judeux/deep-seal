# 00. Product Vision

## 1. 프로젝트 개요

- 작업명: **Project Deep Seal** (가칭)
- 장르: 원정대 운영형 자동 전투 채굴 로그라이트
- 플랫폼: Steam / Windows PC
- 모드: 싱글플레이
- 권장 한 캠페인 길이: 첫 승리 기준 약 8~15시간 **DRAFT**
- 일반 원정 길이: 약 15~20분 **DRAFT**
- 최종 원정 길이: 약 35~50분 **DRAFT**

## 2. 플레이어 판타지

플레이어는 심층 지하를 개척하는 광산 원정대의 관리자이자, 현장에서는 한 명의 광부를 직접 이동시키는 탐험가다.

플레이어가 느껴야 할 핵심 감정은 다음과 같다.

1. 출발 전에 위험을 읽고 준비하는 전략적 긴장
2. 현장에서 예상 밖의 위협을 발견하는 탐험의 긴장
3. 현재 전력으로 싸울지, 우회할지, 후퇴할지를 정하는 판단의 압박
4. 광부가 부상·실종·사망하며 캠페인에 역사가 쌓이는 감정적 무게
5. 여러 원정을 통해 봉인석을 모으고 최종 심층부를 여는 장기적 성취
6. 마지막 핵심 유물을 훔친 뒤 살아 돌아오는 탈출의 절정

## 3. 핵심 차별점

### 3.1 채굴이 전투 지형을 만든다

벽은 자원 상자가 아니라 전술적 공간이다.

- 좁은 통로를 만들어 적을 압축
- 넓은 공간을 파서 회피 공간 확보
- 우회로와 비상 탈출로 개척
- 적의 굴착 경로와 플레이어의 채굴 경로가 충돌
- 내려갈 때 만든 길이 추후 탈출로가 됨

### 3.2 준비와 상성이 난도를 바꾼다

같은 보스라도 준비가 맞으면 수월하고, 준비가 어긋나면 매우 어렵다. 단, 특정 아이템 없이는 절대 승리할 수 없는 하드 카운터는 지양한다.

### 3.3 실패가 새로운 콘텐츠가 된다

원정 실패는 단순 게임오버가 아니다.

- 부상 치료
- 실종자 구조 원정
- 장비 회수 원정
- 네임드 토벌 원정
- 지역 위험도 변화
- 추모비 기록

### 3.4 캠페인은 여러 원정의 누적 결과다

한 판에 봉인석과 최종 목표를 모두 해결하지 않는다. 여러 지역을 조사하고 준비해 봉인석을 확보한 뒤 최종 심층부를 연다.

## 4. 타깃 플레이어

- Vampire Survivors 계열의 빌드 성장과 화면 장악감을 좋아하는 플레이어
- Darkest Dungeon 계열의 장기 원정대 관리와 손실 서사를 좋아하는 플레이어
- 절차적 던전 탐험과 희귀 보물 수집을 좋아하는 플레이어
- 완전한 액션 숙련보다 준비와 판단으로 난도를 낮추는 플레이를 선호하는 플레이어

## 5. 비목표

다음은 현재 게임이 목표하지 않는 경험이다.

- 정밀 조준과 콤보 중심의 액션 게임
- 복잡한 농사·건축·자동화 샌드박스
- 대규모 마을 경영 시뮬레이션
- 갈증·허기·체온·산소를 모두 관리하는 생존 시뮬레이션
- PvP 또는 협동 멀티플레이
- 물과 용암을 셀 단위로 실시간 시뮬레이션하는 유체 게임
