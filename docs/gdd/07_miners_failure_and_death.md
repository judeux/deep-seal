# 07. Miners, Failure, and Death

## 1. 원정대 구조 — DRAFT

- 초기 광부 수: 3명 후보
- 일반 보유 한도: 8~12명 후보
- 한 원정 출전 인원: 1명 — LOCKED
- 후방 지원 인원: 0~2명 후보

## 2. 광부 구성

### 직업

채굴 방식과 전투 성향을 결정한다.

후보:

- 굴착꾼
- 폭파공
- 측량사
- 유물 사냥꾼
- 공명술사

### 특성

장점과 단점을 함께 제공한다.

예:

- 어둠 적응
- 탐욕
- 폐소공포
- 돌귀
- 지병
- 생존 본능

### 경력

원정 생존과 성과로 성장한다. 단순 능력치 증가보다 새로운 시작 선택과 전문화를 해금하는 방향을 우선한다.

### 부상과 변이

원정 실패, 높은 공명 노출, 특정 사건으로 발생한다.

- 신체 손상
- 감각 이상
- 심층부 변이
- 장점과 단점이 함께 있는 변화

## 3. 실패 결과 — LOCKED

일반 원정 실패 시 우선적으로 다음 결과 중 하나 이상이 발생한다.

- 부상 귀환
- 장비·보물 상실
- 장기 치료
- 부정적 특성 또는 변이
- 광부 실종

## 4. 실종 — LOCKED

- 실종자는 해당 지역에 남음
- 일정 원정 횟수 또는 시간 내 구조 원정 생성
- 구조 성공 시 광부와 일부 장비 회수
- 구조 실패 시 영구 사망
- 구조된 광부는 중상 또는 변이를 가질 수 있음

## 5. 영구 사망 — LOCKED

영구 사망은 일반 실패의 기본값이 아니다.

주요 발생 조건:

- 실종 후 구조 실패
- 봉인석 원정의 치명적 실패
- 최종 원정 실패
- 명시된 최고 위험 계약
- 중상 광부를 무리하게 재출전
- 비상 탈출을 거부하고 위험을 계속 감수
- 특정 치명적 저주 또는 사건

### 원칙

- 플레이어가 위험을 사전에 인지할 수 있어야 한다.
- 자동 공격의 타깃 선택 문제 때문에 억울한 즉사를 피한다.
- 영구 사망은 캠페인 역사와 추모 요소를 남긴다.

## 6. 후방 지원 — DEFERRED

출전하지 않는 광부가 제공할 수 있는 보조 효과 후보:

- 측량 지원
- 정비 지원
- 의료 지원
- 운송 지원
- 구조 대기

MVP에서는 제외하고 캠페인 깊이가 부족할 때 추가 검토한다.

## 7. 추모와 기록

사망한 광부는 추모비에 기록한다.

기록 후보:

- 이름과 직업
- 생존 원정 수
- 확보한 주요 보물
- 사망 원인
- 마지막 원정
- 구조 시도 여부
