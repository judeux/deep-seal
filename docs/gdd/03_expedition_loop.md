# 03. Expedition Loop

## 1. 원정 전 단계 — LOCKED

플레이어는 다음 정보를 검토한다.

### 확정 정보

- 지역 및 지층 종류
- 임무 목표
- 주요 적 계열
- 예상 지형 경도
- 알려진 환경 위험
- 예상 보스 또는 흔적

### 불확실한 정보

- 네임드 엘리트 출현 여부
- 정확한 보스 변형
- 서브 던전 종류
- 특수 저주와 사건
- 희귀 보물·광맥 위치

### 준비 항목

- 출전 광부
- 시작 무기 및 채굴 도구
- 방어 장비
- 소모품
- 후방 지원
- 비상 탈출 수단

## 2. 원정 중 핵심 루프 — LOCKED

```text
탐색 → 채굴 → 자동 전투 → 자원 회수 → 성장 선택
      → 위험 징후 확인 → 경로 수정 → 목표 수행
      → 추가 탐욕 또는 귀환 판단
```

## 3. 원정의 주요 결정

- 어느 방향으로 파고 들어갈 것인가
- 전투 공간을 좁힐지 넓힐지
- 희귀 보물을 위해 위험을 감수할지
- 서브 던전에 진입할지
- 네임드 엘리트와 싸울지 우회할지
- 현재 빌드를 보완할지 기존 강점을 극대화할지
- 목표 달성 후 즉시 귀환할지 더 탐사할지

## 4. 원정 유형 — DRAFT

### 채굴 원정

목표 광물을 확보한다. 운반 능력과 장기 생존이 중요하다.

### 조사 원정

여러 조사 지점을 방문한다. 이동성과 탐지가 중요하다.

### 구조 원정

실종 광부를 제한된 조건 내에 구조한다. 속도와 탈출 안정성이 중요하다.

### 토벌 원정

확인된 네임드 또는 보스를 처치한다. 상성 준비가 핵심이다.

### 회수 원정

잃어버린 장비·보물·유해를 회수한다.

### 봉인석 원정

캠페인 주요 임무. 긴 원정과 탈출전이 포함된다.

### 최종 원정

핵심 유물을 확보하고 캠페인의 최종 탈출을 수행한다.

## 5. 성공과 부분 성공

원정은 이분법적 성공/실패만 사용하지 않는다.

- 완전 성공: 목표 달성, 광부 귀환, 주요 보상 확보
- 부분 성공: 광부 귀환, 목표 일부 달성 또는 보상 일부 상실
- 전략적 후퇴: 목표 미달성, 광부와 일부 자원 보존
- 실종: 광부와 장비가 현장에 남음
- 치명적 실패: 고위험 조건에서 사망 또는 지역 악화

## 6. 탈출 단계

일반 원정에서도 귀환 구간이 존재할 수 있으나, 봉인석과 최종 원정에서는 별도의 강화된 탈출전을 사용한다.

- 적 밀도 상승
- 일부 경로 봉쇄
- 네임드 추격
- 운반 보물에 따른 이동 페널티
- 미리 확보한 지름길과 채굴 경로 활용
