# 06. Sub-Dungeons

## 1. 역할 — LOCKED

일반 원정 중 무작위로 발견되는 선택형 소규모 던전이다. 짧고 강한 규칙 변화를 제공하며, 위험을 감수하면 보스 상성 보완·희귀 보물·캠페인 해금을 얻을 수 있다.

## 2. 기본 구조 — DRAFT

- 원정당 0~2개 등장
- 진입은 선택
- 별도 소규모 맵으로 이동
- 플레이 시간 약 3~7분
- 하나의 명확한 핵심 규칙
- 고유 보상 풀
- 일부 입구는 제한 시간 후 폐쇄 가능

## 3. 설계 원칙

- 단순히 적과 보상이 많은 방으로 만들지 않는다.
- 한 문장으로 규칙을 설명할 수 있어야 한다.
- 현재 빌드의 장단점을 시험하거나 보완해야 한다.
- 실패해도 메인 원정 전체가 즉시 끝나지 않는 경우를 우선한다.
- 전용 에셋 요구량을 최소화하고 기존 타일·적·VFX를 재조합한다.

## 4. 서브 던전 템플릿

```text
이름:
한 문장 규칙:
출현 조건:
예상 소요 시간:
핵심 위험:
플레이어의 주요 결정:
현재 빌드와의 상호작용:
보상:
실패 결과:
필요한 전용 콘텐츠:
재사용 가능한 기존 콘텐츠:
상태: DRAFT / LOCKED / DEFERRED
```

## 5. 후보 서브 던전

### 폐수 처리장 — DRAFT

- 규칙: 제한 시간 내 막힌 배수 장치를 열지 않으면 독성 안개가 단계적으로 확산
- 복잡한 물 흐름 대신 구역 단위 위험 증가 사용
- 보상: 독성 저항, 산성 계열 대응 수단

### 납골 저장고 — DRAFT

- 규칙: 벽을 많이 파괴할수록 보상과 적이 함께 증가
- 보상: 생명력, 부활, 사망 관련 유물

### 냉각 동굴 — DRAFT

- 규칙: 일정 시간 같은 구역에 머물면 동결 누적
- 보상: 냉기 공격, 감속 저항, 고열 지형 대응

### 메아리 광산 — DRAFT

- 규칙: 채굴할수록 적에게 위치가 노출되고 웨이브가 강해짐
- 보상: 공명·음파·탐지 관련 장비

### 폐쇄된 승강기 — DRAFT

- 규칙: 부품을 채굴하고 수리하는 동안 방어전 수행
- 보상: 캠페인 지름길 또는 원정 시작 보너스

### 탐식자의 보물고 — DRAFT

- 규칙: 더 많은 보물을 집을수록 출구 수호자가 강해짐
- 보상: 희귀 보물, 운반 관련 유물

### 거울 수정실 — DEFERRED

- 규칙: 현재 빌드의 공격 특성을 복제한 적과 전투
- 밸런스 위험이 높으므로 프로토타입 이후 검토

## 6. 보스 대응 구제 장치

서브 던전은 무작위 보너스 외에도 현장 대응 수단이 될 수 있다.

예:

- 장갑 보스 발견 → 산성 균사굴에서 부식 효과 확보
- 군체 보스 발견 → 화염 보급고에서 광역 연소 효과 확보
- 고속 보스 발견 → 냉각 동굴에서 감속 장비 확보

이 구조는 준비 부족을 완전히 무효화하지 않고, 추가 위험을 감수해 보완할 기회를 제공한다.
