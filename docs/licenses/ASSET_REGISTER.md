# Asset Register

이 문서는 프로젝트에 사용된 외부 에셋, 구매 에셋, 무료 에셋, AI 생성물의 출처와 라이선스를 기록한다.

원칙:
- 프로젝트에 파일을 넣는 순간 이 문서에 기록한다.
- 상업적 이용 가능 여부가 불명확한 에셋은 사용하지 않는다.
- 외부 에셋 원본은 가능한 한 `Assets/ThirdParty` 또는 `ArtSource`에 보관한다.
- 수정본은 원본과 구분해서 기록한다.
